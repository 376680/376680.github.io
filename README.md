# 小鼠联邦调查局 (Mouse FBI)

一个模仿FBI官网设计的小鼠主题网站，使用HTML和CSS构建。

## 更新时间

2024-12-01

## 访问方式

您可以通过以下方式访问这个网站：

1. **本地访问**：在项目根目录下运行 `python3 -m http.server 8080`，然后在浏览器中访问 `http://localhost:8080`
2. **GitHub Pages**：访问 `https://376680.github.io`

## 网站结构

- **导航栏**：包含网站主要导航链接
- **英雄区域**：显示"MISSION FIRST | 使命优先"
- **我想做**：展示各种功能入口
- **最高通缉**：展示不同类型的通缉犯
- **特色内容**：展示新闻、播客等内容
- **领导层**：展示领导团队
- **页脚**：包含网站导航链接

## 技术栈

- HTML5
- CSS3
- 响应式设计

## 设计风格

- 模仿FBI官网的黑色、蓝色、白色配色方案
- 清晰的网格布局
- 悬停效果和动画
- 适配不同屏幕尺寸的响应式设计
